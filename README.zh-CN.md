# 个人经历复盘 Prompt 模板

一个结构化的 AI Prompt 模板，通过"访谈 + 撰写"两阶段流程，帮你完成深度的个人经历复盘。

[English](README.md)

## 这是什么？

这是一个 Prompt 模板，帮你系统性地回顾和反思人生中的任意一段时期——职业发展、创业历程、学习成长、年度总结等。

它不是简单地让 AI "帮我写一篇复盘"，而是把 AI 变成一个专业的访谈者，逐段追问你的经历，最终产出一篇包含三个部分的完整复盘文档：

1. **我的故事** — 以第一人称按时间顺序完整叙述你的经历
2. **旁观者的点评** — 以客观视角分析你的模式、优势和短板
3. **待解决的问题** — 按优先级排列的问题清单，附带具体可行的行动建议

## 工作原理

模板分两个阶段：

### 第一阶段：访谈

你提供一个粗略的起点（时间线 + 简要描述），AI 会逐段追问以深入了解：

- 具体做了什么（实际的项目、行动、产出）
- 为什么做这件事（前因、动机、契机）
- 结果怎样（成果、反馈、得失）
- 当时的主观感受和思考
- 不同事件之间的因果关系和转折点

AI 每次只问一个方向，不会让你应接不暇。当一段问清楚后，再进入下一段。

### 第二阶段：撰写

访谈全部完成后，AI 产出完整的复盘文档，包含上述三个部分。

## 快速开始

1. 打开 [`prompt/zh-CN.md`](prompt/zh-CN.md)（或 [`prompt/en.md`](prompt/en.md) 英文版）
2. 复制其中的 **系统指令** 部分
3. 粘贴到新的 AI 对话开头
4. 在后面跟上你自己的开头叙述，例如：

> 我想复盘自己过去一年的职业发展。起点是2025年6月，我当时刚从公司A离职，原因是……之后我花了两个月找工作……9月入职了公司B……你可以先从这些开始追问我，问完之后我继续讲后面的事。

**提示**：如果你使用 Claude Projects，可以把系统指令粘贴到 Project 的 Custom Instructions 中，每次新建对话都会自动生效。

## 设计理念

- **访谈优先**：AI 不会急于下结论，而是先通过提问了解你的完整故事
- **每次只问一个问题**：不会让你应接不暇，始终聚焦
- **客观直接**：不过度安慰或吹捧，追求诚实的分析
- **可执行的输出**：每个指出的问题都附带具体的下一步行动建议
- **按优先级排序**：底层问题 > 高杠杆问题 > 具体技能差距

## 兼容的 AI 工具

- DeepSeek
- 以及其他类似的 AI 助手
